ГОТОВАЯ СТРУКТУРА ДЛЯ GITHUB PAGES

Загрузите СОДЕРЖИМОЕ этой папки в корень репозитория, не саму внешнюю папку.

В корне GitHub должны сразу находиться:
- index.html
- menu.css
- .nojekyll
- папка assets
- папка dropfish
- папка agroclick

DropFish уже полностью размещён в папке dropfish:
- dropfish/index.html
- dropfish/style.css
- dropfish/script.js

В index.html DropFish стоят версии кэша 14.19:
- style.css?v=14.19
- script.js?v=14.19

ВАЖНО:
1. Не переименовывайте menu.css в style.css.
2. Не создавайте дополнительную вложенную папку github-pages-ready-v14.19 на GitHub.
3. Для AgroClick замените файл README-PUT-AGROCLICK-HERE.txt реальными файлами игры.
4. Итоговый путь игры должен быть agroclick/index.html.
